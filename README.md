# Weather Or Not: A Weather Dashboard

## Purpose ##
An online weather application that allows you to gather an immediate-upcoming 5 day forecast of any searched city. 

## Built With ##
* JQuery
* CSS
* Bootstrap
* OpenWeatherMap API
* HTML
